# Bottle 3D Viewer - Practice Project

This is a practice project featuring an interactive 3D bottle viewer using Spline technology.

## Description

A web-based application that displays a 3D bottle model with an interactive viewer powered by Spline. The project includes a navigation bar and a content overlay with call-to-action button.

## Technologies Used

- HTML5
- CSS3
- Spline Viewer (3D Interactive)
- JavaScript (Module-based)

## Features

- Responsive navigation bar
- Interactive 3D model rendering
- Smooth animations and transitions
- Content overlay with explore button

## Purpose

This repository is created for **learning and practice purposes only**. It's used to explore web technologies, 3D visualization, and UI/UX design concepts.

## Getting Started

1. Clone the repository
2. Open `bottle.html` in a modern web browser
3. Interact with the 3D bottle model
